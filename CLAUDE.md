# OddSockets C SDK
C99 SDK for embedded/IoT real-time messaging. libcurl + libwebsockets.
Key files: `src/oddsockets.h`, `src/oddsockets.c`, `src/manager_discovery.c`, `src/websocket_client.c`, `src/json_parser.c`, `src/http_client.c`.
Free tier: 100 MAU, 50 connections, 10K msg/day, 10 channels, 100MB/24h.
